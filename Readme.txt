Данное мини-приложение создано для удобства ведения учета домашних расходов. 
Данные сохраняются и подсчитываются локально, без использования сторонних сервисов.
Написано на нативном JavaScript, Node.js и собрано с помощью Electron.
Стартовый файл app.js
Предполагает установленную на компьютере Node.js
